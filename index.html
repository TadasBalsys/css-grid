Row 12

min-content, max-content, minmax()

minmax(minimum cells size, maximum cell size), e.g. grid-template-rows: minmax(95px, min-content) 250px; First row minimum height must be 95px, maximum height - so much as it can fit content. The second row height is set to 250px, thats way not all content of cell nr: 7 is visible. If the second row will be set to min-content, e.g. grid-template-rows: minmax(95px, min-content) min-content; The second row will expend its height to show all content in grid cells(in this example, cell nr: 7 is guilty for large seconds row height)

Row 13

auto-fit and auto-fill

  • grid-template-rows: repeat(auto-fit, minmax(100px, 1fr)) - created as much rows that needed to fit content. rows minimum height is set to 100px, maximum 1fr(grow as much as needed to fit content)
  • grid-template-columns: repeat(auto-fit, minmax(220px, 1fr)) - created columns(8, 7 or 5 columns, etc) as much as needed to fit content. minimum width of the column must be 220px, maximum 1fr(grow as much as needed to fit content)
  • grid-auto-rows: 350px; - rows which are generated automatically to fit content (which is no longer fitting in the declared rows), must have height 350px.
